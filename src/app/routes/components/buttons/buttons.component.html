Button Variations

Basic Buttons

Add .btn .btn-[variation] to any <button> or <a> for stylized buttons. Button variations are shown below:

Outline Buttons

Use .btn-outline-[variation] on any button variation for outline only button

Pill Buttons

Add .btn-pill to any button variation to use buttons with round sides

Flat Buttons

Use .btn-flat-[variation] on any button variation for flat material style button

FAB Buttons

Add .btn-fab to any button variation to use round action button
