Buttons

Simplify Admin expands the already great button .btn variations provided by bootstrap to include even more button styles and colors.

It also provides utilities to make your life easier.

Button Variations
Basic Buttons

Add .btn .btn-[variation] to any <button> or <a> for stylized buttons. Button variations are shown below:

Outline Buttons

Use .btn-outline-[variation] for outline only button

Flat Buttons

Use .btn-flat-[variation] for flat material style button

Pill Buttons

Add .btn-pill to any button style variation (normal, flat or outline) to use buttons with round sides

Example .btn-pill on outline buttons

FAB Buttons

Add .btn-fab to any button variation to use round action button
