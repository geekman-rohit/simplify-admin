Progress

Simplify Admin expands the already great Progress .progress variations provided by bootstrap to include even more styles and colors.

We recomment using thin material progress bars

Progress Variations
Progress

Use a .bg-[variation] class to generate a colored progress bar

Use a .bg-primary class to Primary progress bar

Use a .bg-success class to Success progress bar

Use a .bg-info class to Info progress bar

Use a .bg-danger class to Danger progress bar

Use a .bg-warning class to Warning progress bar

You can also use .bg-dark, .bg-secondary, .bg-light classes

Thin Progress Bars

Use a .progress-thin or .progress-sm with .progress for a material style.
