Timeline Classes
Class Use
.timeline Wrapper Element Around Your timeline
.timeline-lg Add this class to .timeline to generate bigger dots
.timeline-event A timeline event wrapper, add this around each timeline event
.timeline-event-[[color]] Add a color variant to .timeline-event where color is one of the theme colors.
.timeline-line Add Inside a .timeline-event to add a line on the left
.timeline-event-card Wrapper around the timeline text
.single-dot Add this class to a .timeline-event to show a single dot instead of a default dot inside cirlce
.single-circle Add this class to a .timeline-event to show a single circle instead of a default dot inside cirlce
